Ingeniero de Software

Tech Solutions Inc. | Remoto

Descripción del Puesto

Estamos buscando un Ingeniero de Software talentoso y motivado para unirse a nuestro equipo dinámico. El candidato ideal tendrá experiencia en desarrollo de software, habilidades para resolver problemas y una pasión por la tecnología.

Responsabilidades:

  • Desarrollar, probar y mantener aplicaciones de software de alta calidad.
  • Colaborar con equipos multifuncionales para definir, diseñar y enviar nuevas características.
  • Identificar y corregir cuellos de botella y errores en el software.
  • Mantenerse actualizado con las últimas tendencias y tecnologías en desarrollo de software.

Requisitos:

  • Título en Ciencias de la Computación o campo relacionado.
  • Experiencia comprobada como Ingeniero de Software o rol similar.
  • Fuerte conocimiento de lenguajes de programación como Java, Python o C++.
  • Habilidades sólidas de resolución de problemas y pensamiento analítico.
  • Capacidad para trabajar de manera efectiva en un entorno de equipo colaborativo.

Beneficios:

  • Salario competitivo y paquete de beneficios.
  • Oportunidades de desarrollo profesional y crecimiento dentro de la empresa.
  • Ambiente de trabajo flexible y remoto.
